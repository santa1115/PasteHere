# Paste Here for iOS

Since iOS doesn't have a "Mouse Right-Click", we use the **Share Sheet** or a **Home Screen Widget**.

1.  Open the **Shortcuts** app on your iPhone/iPad.
2.  Tap **+** to create a new shortcut.
3.  Name it **"Paste & Download"**.
4.  Add the following actions:

    *   **Get Clipboard**
    *   **Get Contents of URL** (Method: GET)
        *   *Connect this to the Clipboard output*
    *   **Save File**
        *   *Turn off "Ask Where to Save" if you want it to save to a specific folder automatically.*
