# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 3.3.x   | :white_check_mark: |
| < 3.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within PasteHere, please do **not** open a public issue.
Instead, please email the maintainer directly or create a private GitHub Advisory draft.

We will try to address the issue as soon as possible.
